67 Blankets Mandela Day Web Application


Overview
The 67 Blankets Mandela Day Web Application is designed to support the 67 Blankets Mandela Day initiative. It allows users to sign up, browse events, make donations, and place orders. The web app integrates Firebase for backend services and serves as a complementary platform to the mobile app.

Features
User Authentication: Sign up and login functionality for both general users and admins.
Ambassador Form: Users can apply to become ambassadors by filling out a form.
Donation Form: Users can submit donation forms to contribute to the cause.
Event Management: View and register for upcoming events.
Order System: Browse, select, and purchase products from the online store.
Admin Dashboard: Admin users can manage forms, orders, and user data through a secure panel.

Technologies Used
ASP.NET Core: Backend framework for creating a dynamic web application.
Firebase: Used for authentication and storing data with Firestore.
C#: Primary programming language for backend development.
HTML/CSS: For structuring and styling the pages.
JavaScript: For dynamic content and form validation.
Bootstrap: Frontend framework used to design responsive pages.
Visual Studio 2019: Integrated Development Environment (IDE) for building and testing the web application.

Requirements
ASP.NET Core Runtime.
Visual Studio 2019 or higher.
Firebase Project for authentication and database services.

Installation Instructions
Clone the repository or download the project as a ZIP file.
Open Visual Studio 2019.
Select File > Open and navigate to the project directory.
Ensure that you have Firebase integrated into your project. Follow the setup guide here: Firebase Setup for Web.
Build the project and run it locally using the Visual Studio development server or host it on your preferred server.

Firebase Configuration
Set up Firebase Authentication for user and admin login functionalities.
Use Firestore to manage data related to ambassador forms, donations, and orders.
Ensure that the Firebase project is linked to both the mobile and web applications.

How to Use
Login: Sign in through the login page using your credentials as a user or admin.
Browse and Register for Events: Check out the events page to see upcoming events and register for participation.
Place an Order: Add items to the cart and complete your purchase through the order page.
Submit Forms: Fill out the ambassador or donation forms to support the cause.
Admin Panel: Admin users can access the dashboard to view and manage submissions.
